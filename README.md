# 🔐 Password Generator & Strength Checker (GUI)

## 🚀 Overview

A simple Python GUI application that allows users to:

* Check password strength
* Generate strong random passwords

Built using Tkinter.

---

##  Features

* Password strength checker (Weak / Medium / Strong)
* Random secure password generator
* Simple and clean GUI interface

---

## 🛠️ Tech Stack

* Python
* Tkinter
